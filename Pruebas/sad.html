Enlaces

Etiqueta enlace -> a href=""
<href="#"> es una practica abitual, recarga la pagina
<href="direccion a enlazar">para poner un enlace podemos usar la etiqueta junto a
<a href="..." target="_blank"> te abre en una pestaña nueva
<a href="..." target="_self"> es el que viene default, se quedda en la pestaña
<a href="index2.html"> podemos enlazar a un archivo local
<a href="#sitioId"> podemos enlazar a un apartado del html
<h1/p... id="sitioId"> tenemos que indiar allí donde queramos enlazar
<a href="/index.html"> podemos abrir otro documento en cualquier direccion
<ahref="../../Pruebas/index.html">podemos tanto retroceder como avanzar enlos repositorios

<img src="../images/hola.jpg>
alt="hola" -> etiqueta obligatoria para cuando hay un problema con la presentacion de la imagen, vemos el texto
height="155" -> ajusta la altura
width="150" -> ajusta el ancho de la imagen, si se cambia solo el ancho, la altura se ajusta (con width: numero% se ajusta al porcentaje de la pantalla que se indique)

Tablas

Para definir tabla se usa la etiqueta <tabla>, dentro de ella están las filas <tr> y luego las celdas <td>
<tborder="nºpx"> atributo dentro de la etiqueta tabla, se definen bordes
<th> en lugar de td define el caracter del la celda como encabezado
<Caption> se usa para el titulo de la tabla </caption>
Al igual que en un codigo html se puede dividir la tabla <thead> en head y body usando <tbody>
<table width="nº"> para definir el ancho de la celda, se puede usar tambien en % para que se extienda el porcentaje de la página deseado
